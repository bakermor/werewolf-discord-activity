# Discord One Night Werewolf Activity

A Discord Activity version of **One Night Werewolf**, a social deduction party game. Play 3-5 players entirely within Discord, with night-phase role actions, day-phase discussion, and voting.

## Upcoming Features

- Discord-based lobby with 3-5 players
- Available Roles: Werewolf, Seer, Robber, Troublemaker, Villager
- Real-time night phase orchestration
- Day-phase discussion and voting
- Win condition and results display
- Play again option

## Technologies

- React (Frontend)
- Node.js / Express with TypeScript (Backend)
- WebSockets (Real-time communication)
- Discord API / Activity SDK

## Setup

1. Clone the repository
2. Install dependencies
3. Configure environment variables (VITE_DISCORD_CLIENT_ID, DISCORD_CLIENT_SECRET). For detailed instructions, see the [Discord Activity documentation](https://discord.com/developers/docs/activities/building-an-activity#step-2-creating-an-app)
4. Run frontend and backend
